# YouTube Info API

Simple YouTube channel info API deployed on Vercel.

## Endpoint

/api/yt-info?channel=@username

## Example

https://your-api.vercel.app/api/yt-info?channel=@MrBeast

## Response

{
  "channel_title": "MrBeast",
  "channel_id": "UCX6OQ3DkcsbYNE6H8uQQuVA",
  "handle": "@MrBeast",
  "subscribers": "240M subscribers",
  "views": "45B views",
  "videos": "800",
  "published_at": "Feb 20, 2012"
}
